# Descodificando-DNA
**Vamos a proporcionar herramientas para descifrar el DNA**

*El propósito de este repositorio es poder ayudar a principiantes a descifrar el DNA: leer secuencias de DNA, transcribirlas a secuencia de RNA y traducirlas a proteínas.*

El repositorio se encuentra organizado en 3 carpetas: 01_Data, 02_scripts y 03_results/protein_output.
En 01_Data se encuentra el documento formato .txt con la secuencia de ADN a analizar. En este ejemplo, la secuencia de ADN del gen humano BRCA2.
En 02_scripts se encuentran los distintos scripts en formato .ipynb para descifran el DNA. A su vez, se encuentran organizados en base_count, complementary,transcription y translation, para una mejor organización.
En 03_results/protein_output se encuentra la secuencia de aminoácidos en formato .txt, ya que se trataría del resultado final obtenido a partir de los scripts anteriores.
