# Descodificando-DNA
**Vamos a proporcionar herramientas para descifrar el DNA**

*El propósito de este repositorio es poder ayudar a principiantes a descifrar el DNA: leer secuencias de DNA, transcribirlas a secuencia de RNA y traducirlas a proteínas*

Colaboradores: Abdelfatah Oumouss Aitmoussa, Alex Ibañez de Fez, Andrea Polo Martin, Carla Garcia Pitarch y Carlos Fernandez Castilla.
